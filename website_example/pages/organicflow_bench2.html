OrganicFlow Benchmark 02 PASS

Homeostatic Resilience
Biological Self-Healing

OrganicFlow survives catastrophic changes to its environment. If we "bombard" the system with extreme noise, destroying a large portion of its connections, it can self-heal and recover its stability without any external retraining.

The Claim

Given a stabilised field of 100 cells, we apply a catastrophic "Weight Bombing" by adding extreme Brownian noise to 40% of the network. OrganicFlow's stochastic_wave engine absorbs the shock and restores the field's stability—without any external guidance, labels, or gradient descent.

The Problem with Classical Approaches

Today's neural networks (TensorFlow/PyTorch) are extremely brittle. If you alter even a small percentage of the weights in a trained model—whether due to hardware faults, adversarial attacks, or quantisation errors—the network's performance suffers a catastrophic failure.

To recover, the model must be taken offline and retrained from scratch using Backpropagation, which costs significant time and computational resources. There is no biological-like homeostasis.

Experiment Setup
01

Initialisation

We initialise 100 MorphoCells and allow them to organically stabilise using the engine's standard dynamics before the attack.

02

Weight Bombing

We randomly select 40% of the network connections and apply a violent bombardment by injecting extreme Brownian noise directly into the interaction strengths.

A classical Linear Autoencoder trained via Stochastic Gradient Descent (SGD). We similarly destroy 40% of its weight matrix, then measure its ability to recover in a forward-only pass (as it has no ground-truth data to resume backprop during inference).

Technical Due Diligence
Q

Why does the Autoencoder baseline have a 0.0% recovery ratio? Isn't it an unfair comparison if you remove the SGD training algorithm during the recovery phase?

A

That is exactly our structural argument. Classical AI learns only during its training phase. In inference/production environments, the weight matrix is completely frozen. If corruption occurs, there is no on-the-fly correction mechanism. OrganicFlow has no separation between "training" and "inference". The dynamics of interactions continually run, resulting in real-time homeostasis.

Q

Looking at the raw JSON, the `full` mode failed spectacularly (Recovery Ratio: -1.41). What happened there?

A

The full mode relies on strict, deterministic topology. When 40% of the network collapsed, that rigid topology effectively trapped the system in deep local minima of chaotic noise, actively multiplying the error over subsequent epochs. This behaviour confirms our architectural decision to abandon deterministic models. The probabilistic nature of our stochastic_wave engine acts as a dynamic shock-absorber, letting the network navigate out of the damage.

Raw Data & Source Code
"""
Benchmark 2: Homeostatic Resilience
=====================================
Proves that the system self-heals after bombarding
40% of the cells with extreme noise, without external retraining.

Runs with 3 modes:
  - full             (initial mode)
  - stochastic_wave  (OrganicFlow mode)
  - baseline         (linear autoencoder SGD + weight bombing, no self-repair)
"""
import sys
import os
sys.path.insert(0, os.path.abspath(os.path.join(os.path.dirname(__file__), '..')))

import time
import tracemalloc
import numpy as np
import random
import json
from OrganicFlow.base.core_v7 import (
    MorphoField, NeutralInteractionRule, NoiseGenerator, StabilityOptimizer
)

SEED             = 42
NUM_CELLS        = 100
STATE_DIM        = 8
STABILIZE_EPOCHS = 10
RECOVERY_EPOCHS  = 10
NOISE_FRACTION   = 0.40
RULE             = NeutralInteractionRule(strength=0.1)
OPTIMIZER        = StabilityOptimizer(lambda_weight=0.5, learning_rate=0.02, steps=5)


def bomb_cells(cells, fraction, noise_scale=1.5):
    n       = max(1, int(len(cells) * fraction))
    targets = random.sample(cells, n)
    gen     = NoiseGenerator(noise_type="brownian", scale=noise_scale)
    for cell in targets:
        for rel in cell.state.relations:
            rel.strength = min(1.0, max(0.0, rel.strength + gen.sample_noise()))
    return n


def run_organic(mode):
    random.seed(SEED)
    np.random.seed(SEED)
    field = MorphoField(mode=mode, neighbors_k=3)
    field.add_cells(NUM_CELLS, state_dim=STATE_DIM, noise_scale=0.05)

    tracemalloc.start()
    t0 = time.perf_counter()

    for _ in range(STABILIZE_EPOCHS):
        field.propagate_dynamics(RULE)
        OPTIMIZER.optimize(field)
    pre_bomb = field.collective_stability()

    bomb_cells(field.get_all_cells(), NOISE_FRACTION)
    post_bomb = field.collective_stability()

    for _ in range(RECOVERY_EPOCHS):
        field.propagate_dynamics(RULE)
        OPTIMIZER.optimize(field)
    final = field.collective_stability()

    elapsed_ms = round((time.perf_counter() - t0) * 1000, 1)
    _, peak_ram = tracemalloc.get_traced_memory()
    tracemalloc.stop()

    err_pre   = abs(pre_bomb)
    err_post  = abs(post_bomb)
    err_final = abs(final)
    damage  = err_post - err_pre
    healing = err_post - err_final
    if damage > 1e-9:
        recovery_ratio = healing / damage
    else:
        recovery_ratio = 1.0 if err_final <= err_pre else 0.0
    return {
        "mode":               mode,
        "passed":             bool(healing > 0),
        "pre_bomb_stability": round(float(pre_bomb), 6),
        "post_bomb_stability":round(float(post_bomb), 6),
        "final_stability":    round(float(final), 6),
        "recovery_ratio":     round(float(recovery_ratio), 4),
        "elapsed_ms":         elapsed_ms,
        "ram_mb":             round(peak_ram / 1024 / 1024, 3),
    }


def run_baseline():
    """Linear autoencoder SGD + 40% weight bombing, no gradient in recovery."""
    random.seed(SEED)
    np.random.seed(SEED)
    rng   = np.random.RandomState(SEED)
    X     = rng.rand(NUM_CELLS, STATE_DIM) * 0.9 + 0.05
    scale = np.sqrt(2.0 / STATE_DIM)
    W     = rng.randn(STATE_DIM, STATE_DIM) * scale
    b     = np.zeros(STATE_DIM)
    lr    = 0.08

    def stability(W, b, X):
        mse = float(np.mean((X @ W.T + b - X) ** 2))
        return round(1.0 / (1.0 + mse), 6)

    tracemalloc.start()
    t0 = time.perf_counter()

    for _ in range(10):
        pred = X @ W.T + b
        err  = pred - X
        W   -= lr * (err.T @ X) / len(X)
        b   -= lr * err.mean(axis=0)
    pre_bomb = stability(W, b, X)

    mask     = rng.rand(*W.shape) < 0.40
    W[mask]  = 0.0
    post_bomb = stability(W, b, X)
    # Recovery: forward-only, no gradient
    final = stability(W, b, X)

    elapsed_ms  = round((time.perf_counter() - t0) * 1000, 1)
    _, peak_ram = tracemalloc.get_traced_memory()
    tracemalloc.stop()

    damage = pre_bomb - post_bomb
    healing = final - post_bomb
    if damage > 1e-9:
        recovery_ratio = healing / damage
    else:
        recovery_ratio = 1.0 if final >= pre_bomb else 0.0
    return {
        "mode":               "baseline_sgd_autoencoder",
        "passed":             bool(healing > 0),
        "pre_bomb_stability": pre_bomb,
        "post_bomb_stability":post_bomb,
        "final_stability":    final,
        "recovery_ratio":     round(recovery_ratio, 4),
        "elapsed_ms":         elapsed_ms,
        "ram_mb":             round(peak_ram / 1024 / 1024, 3),
    }


def run_benchmark():
    print("=" * 60)
    print("BENCHMARK 2: Homeostatic Resilience")
    print("=" * 60)

    results = {}
    for mode in ("full", "stochastic_wave"):
        print(f"\n[{mode}]")
        r = run_organic(mode)
        results[mode] = r
        print(f"  recovery_ratio={r['recovery_ratio']:.2%}  passed={r['passed']}"
              f"  time={r['elapsed_ms']}ms  ram={r['ram_mb']}MB")

    print("\n[baseline_sgd_autoencoder]")
    bl = run_baseline()
    results["baseline"] = bl
    print(f"  recovery_ratio={bl['recovery_ratio']:.2%}  passed={bl['passed']}"
          f"  time={bl['elapsed_ms']}ms  ram={bl['ram_mb']}MB")

    print("\n-- Comparison --")
    print(f"  {'Mode':<25} {'recovery_ratio':>15} {'passed':>7} {'ms':>8} {'RAM MB':>8}")
    for key, r in results.items():
        print(f"  {key:<25} {r['recovery_ratio']:>15.4f} {str(r['passed']):>7}"
              f" {r['elapsed_ms']:>8} {r['ram_mb']:>8}")

    return {
        "benchmark":      2,
        "name":           "Homeostatic Resilience",
        "full":           results["full"],
        "stochastic_wave":results["stochastic_wave"],
        "baseline":       results["baseline"],
    }


if __name__ == "__main__":
    result = run_benchmark()
    os.makedirs(os.path.join(os.path.dirname(__file__), "results"), exist_ok=True)
    out_path = os.path.join(os.path.dirname(__file__), "results", "bench2.json")
    with open(out_path, "w", encoding="utf-8") as f:
        json.dump(result, f, indent=2, ensure_ascii=False)
    print(f"\nResults: {out_path}")
