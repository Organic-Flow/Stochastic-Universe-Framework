Contextual Epigenetics
Environmental Adaptation

OrganicFlow allows the exact same input data to produce completely different internal representations depending on the broader algorithmic "context", simulating Epigenetics.

The Claim

Given the exact same input data array, OrganicFlow's stochastic_wave engine produces radically different internal variance and topological behaviour based on the environmental context (Threat vs. Strategy). Classical neural networks, in contrast, remain entirely blind to environmental state changes.

The Problem with Classical Approaches

In a classical Neural Network architecture (TensorFlow/PyTorch), the weight matrix W is inherently "context-blind". If you feed it an input vector X, it will deterministically compute the exact same output Y every time.

To force the network to adapt its behaviour (e.g., to act more "conservatively" or "aggressively" based on the environment), you must either append the context as extra input features—bloating the model's complexity—or deploy an entirely new model for each contextual state.

Experiment Setup
01

Data Initialisation

We feed the exact same identical input vector (an array of 0.5 values) into two different isolated MorphoFields.

02

Threat Context (Field 1)

Simulates a high-stakes threat environment. The network induces an epigenetic shift, driving the cells to express increased "noise" and high adaptable variance to explore escape or defence topologies.

03

Strategy Context (Field 2)

Simulates a calm, stable strategic environment. The cells are epigenetically restrained to prioritize absolute precision, resulting in extreme low variance and tightly bound clusters.

04

The Baseline

A classical, fixed-weights matrix multiplication block. We process the identical input through both "contexts" to measure if the network can organically alter its propagation variance.

Epigenetic Contextual Variance

Technical Due Diligence
Q

Why do you call a simple noise scale controller "Epigenetics"? Isn't that a marketing exaggeration?

A

In biology, epigenetics describes how the exact same DNA sequence can be physically expressed in drastically different ways depending on chemical signals originating from the environment. In our architecture, the "DNA sequence" (the initial sensory input values) remains identical, but its spatial manifestation changes globally via Contextual Noise gradients. Think of it as Dynamic Inference: the system can autonomously adjust—on the fly—how every single agent processes data depending on macro-variables, without requiring us to change a single line of the node interaction logic.

Q

Where is this mechanism practically useful in production?

A

It is immediately critical in environments like Robotics, Autonomous Driving, and Algorithmic Trading. A quadruped robot might use the exact same Neural Network layer to control its "walking gait". However, if its sensors deduce it just stepped on ice (activating a Threat Context), its internal data representation must immediately allow for rapid, extreme (high-variance) physical corrections to preserve balance. If it walks on flat asphalt (Strategy Context), it must conserve energy and eliminate jitter (low-variance). With OrganicFlow, you do not need to hot-swap two separate, parallel Models into RAM to achieve this duality.

"""
Benchmark 4: Contextual Epigenetics (MorphoDNA)
================================================
Same input, different context -> completely different internal variance.
Runs with 3 modes: topological | stochastic_wave | baseline fixed-weights
"""
import sys, os, time, tracemalloc, json
sys.path.insert(0, os.path.abspath(os.path.join(os.path.dirname(__file__), "..")))
import numpy as np
from OrganicFlow.base.core_v7 import MorphoCell, MorphoField, NeutralInteractionRule
from OrganicFlow.base.helping.morpho_dna import morpho_dna_pipeline

STATE_DIM=8; EPOCHS=8
CONTEXT_THREAT="Ares_conflict"; CONTEXT_STRATEGY="Athena_strategy"
CONTEXT_NOISE={CONTEXT_THREAT:0.35, CONTEXT_STRATEGY:0.02}
SAME_INPUT=[0.5]*STATE_DIM

def make_cell(traits_values, context):
    ns=CONTEXT_NOISE[context]; traits=[f"feature_{i}" for i in range(len(traits_values))]
    cell=MorphoCell(state_dim=len(traits_values),traits=traits,noise_scale=ns)
    for i,t in enumerate(traits): cell.state.trait_strengths[t]=traits_values[i]
    cell.dna_context=context; cell.encode_dna(context=context)
    return cell

def field_stats(field):
    vecs=np.array([list(c.state.trait_strengths.values()) for c in field.get_all_cells()])
    return float(vecs.var()), float(vecs.mean())

def run_organic(mode):
    np.random.seed(42); rule=NeutralInteractionRule(strength=0.1)
    tracemalloc.start(); t0=time.perf_counter()
    ft=MorphoField(mode=mode,neighbors_k=3)
    for _ in range(20): ft.cells_by_dim[STATE_DIM].append(make_cell(SAME_INPUT,CONTEXT_THREAT))
    for _ in range(EPOCHS):
        ft.propagate_dynamics(rule)
        for c in ft.get_all_cells(): c.sample_noise()
    var_t,mean_t=field_stats(ft)
    fs=MorphoField(mode=mode,neighbors_k=3)
    for _ in range(20): fs.cells_by_dim[STATE_DIM].append(make_cell(SAME_INPUT,CONTEXT_STRATEGY))
    for _ in range(EPOCHS):
        fs.propagate_dynamics(rule)
        for c in fs.get_all_cells(): c.sample_noise()
    var_s,mean_s=field_stats(fs)
    ms=round((time.perf_counter()-t0)*1000,1); _,pr=tracemalloc.get_traced_memory(); tracemalloc.stop()
    ts="".join([chr(65+int(v*25)) for v in SAME_INPUT])
    _,ct,_=morpho_dna_pipeline(ts,CONTEXT_THREAT); _,cs,_=morpho_dna_pipeline(ts,CONTEXT_STRATEGY)
    vd=abs(var_t-var_s); md=abs(mean_t-mean_s); dd=ct!=cs
    return {"mode":mode,"passed":bool(dd or vd>1e-8 or md>1e-8),
            "var_threat":round(var_t,6),"var_strategy":round(var_s,6),
            "var_divergence":round(vd,6),"mean_divergence":round(md,6),
            "dna_outputs_differ":bool(dd),"elapsed_ms":ms,"ram_mb":round(pr/1024/1024,3)}

def run_baseline():
    rng=np.random.RandomState(42); W=rng.randn(STATE_DIM,STATE_DIM)*0.3
    def proc(ns):
        s=np.tile(SAME_INPUT,(20,1)).astype(float)
        for _ in range(EPOCHS): s=(np.tanh(s@W.T)+1)/2
        return float(s.var()),float(s.mean())
    tracemalloc.start(); t0=time.perf_counter()
    vt,mt=proc(0.35); vs,ms2=proc(0.02)
    elapsed=round((time.perf_counter()-t0)*1000,1); _,pr=tracemalloc.get_traced_memory(); tracemalloc.stop()
    vd=abs(vt-vs); md=abs(mt-ms2)
    return {"mode":"baseline_fixed_weights","passed":bool(vd>1e-8 or md>1e-8),
            "var_threat":round(vt,6),"var_strategy":round(vs,6),
            "var_divergence":round(vd,8),"mean_divergence":round(md,8),
            "note":"Fixed W: context-blind — var_divergence ~0",
            "elapsed_ms":elapsed,"ram_mb":round(pr/1024/1024,3)}

def run_benchmark():
    print("="*60); print("BENCHMARK 4: Contextual Epigenetics"); print("="*60)
    results={}
    for mode in ("topological","stochastic_wave"):
        print(f"\n[{mode}]"); r=run_organic(mode); results[mode]=r
        print(f"  var_div={r['var_divergence']:.2e}  dna_diff={r['dna_outputs_differ']}  passed={r['passed']}  time={r['elapsed_ms']}ms")
    print("\n[baseline_fixed_weights]"); bl=run_baseline(); results["baseline"]=bl
    print(f"  var_div={bl['var_divergence']:.2e}  passed={bl['passed']}  ({bl['note']})")
    return {"benchmark":4,"name":"Contextual Epigenetics",
            "topological":results["topological"],"stochastic_wave":results["stochastic_wave"],"baseline":results["baseline"]}

if __name__ == "__main__":
    result=run_benchmark()
    os.makedirs(os.path.join(os.path.dirname(__file__),"results"),exist_ok=True)
    out=os.path.join(os.path.dirname(__file__),"results","bench4.json")
    with open(out,"w",encoding="utf-8") as f: json.dump(result,f,indent=2,ensure_ascii=False)
    print(f"\nResults: {out}")
