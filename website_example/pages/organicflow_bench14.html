XOR Expressivity
Solving The Problem That Froze AI

OrganicFlow achieves the classification of non-linearly separable datasets (such as the classic XOR problem and "Two Moons") without using Hidden Layers, Activation Functions, or Backpropagation. Non-linearity is not achieved by "warping" the space mathematically, but by dynamically moving the data itself within it.

The Claim

Deep Learning enforces non-linearity by adding complex hidden layers and Activation functions. OrganicFlow shatters this rigid paradigm. Instead of mathematically warping the coordinate space to fit the data, it dynamically moves the data itself within the space, transforming complex non-linear problems into trivial linear ones.

The Problem with Classical Approaches

In 1969, Marvin Minsky and Seymour Papert proved in "Perceptrons" that a simple, single-layer neural network (a linear classifier) is mathematically incapable of solving the XOR problem. This undeniable proof triggered the devastating first "AI Winter," freezing global investment for almost two decades. The computational brute-force solution, which arrived 20 years later and remains standard today, is to inject matrix Hidden Layers coated in non-linear activation functions (ReLU/tanh) strictly to arbitrarily "distort" the active spatial dimensions until data lines up linearly.

Pre-emptive Q&A (Technical Due Diligence)
Q

So conceptually, you are essentially functioning as a "pre-processing step" designed to make the data easier for standard ML classifiers?

A

Exactly. However, this "pre-processing step" is the core model itself! Rather than adding massive complexity to the classifier via hidden layers and activations, we keep the classifier primitive (a pure linear "brain") and place the complexity directly into the representation of the data. It is a fundamentally different philosophy. Think of it as a dynamic "breathing" embedding space that autonomously reconfigures itself to solve the problem.

Q

This benchmark represents a Supervised scenario (you actively use labels to attract or repel cells). Doesn't this contradict OrganicFlow's Unsupervised nature?

A

Not at all. OrganicFlow operates as a mathematically "agnostic" framework. The spatial interaction rules act as plug-and-play logic modifiers. These rules can be purely Unsupervised (as proven extensively in Benchmarks 1, 5, and 7) or explicitly Supervised (like in this benchmark). Crucially, the foundational mechanism—topological cellular rearrangement—remains exactly the same. This proves the architecture's inherent flexibility to solve perfectly both structure discovery (Unsupervised) and task-driven classification (Supervised) problems using the same spatial physics.

"""
Benchmark 14: XOR Expressivity -- Topological Non-Linearity
=====================================================================
Proves that OrganicFlow solves XOR without Hidden Layers,
without ReLU, without backpropagation.

History: In 1969, Minsky proved that a Perceptron cannot solve
XOR -- the problem is NOT linearly separable in the initial space.
This "froze" AI research for ~20 years.

The classical solution: Hidden Layer + Non-Linear Activation (ReLU/tanh).
The organic solution: cells rearrange their own world
  via topological attraction/repulsion -- the space becomes linearly separable.

Protocol:
  4 MorphoCells in 2D space (XOR corners):
    (0,0) label=0   (0,1) label=1
    (1,0) label=1   (1,1) label=0
  FeaturePullInteractionRule: same-label -> attract, diff-label -> repel.
  After epochs, LogisticRegression on new positions -> accuracy 1.0.

Two Moons Extension:
  200-point two-moons, supervised rule with moon labels.
  Baseline LogReg ~0.88 (non-linear problem).
  Organic: rearranges the points -> linearly separable.

Pass Criteria:
  xor organic_logreg_final == 1.00  (perfect XOR solution)
  xor baseline_logreg_raw  <= 0.75  (proves non-linearity)
  moons organic_logreg > baseline + 0.05
"""
import sys
import os
sys.path.insert(0, os.path.abspath(os.path.join(os.path.dirname(__file__), "..")))

import json
import time
import tracemalloc
import numpy as np

from OrganicFlow.base.core_v7 import MorphoCell, MorphoField, FeaturePullInteractionRule

SCRIPT_DIR  = os.path.dirname(os.path.abspath(__file__))
RESULTS_DIR = os.path.join(SCRIPT_DIR, "results")

# ─── Config ───────────────────────────────────────────────────────────────────

SEED              = 42
XOR_EPOCHS        = 120
XOR_CELLS_CORNER  = 8       # cells per XOR corner (noise spread, same label)
XOR_NOISE         = 0.04    # within-corner spread
MOONS_EPOCHS      = 60
MOONS_N           = 200
MOONS_NOISE       = 0.15
LEARNING_RATE     = 0.12
NEIGHBORS_K       = 4


# ─── Cell helpers ─────────────────────────────────────────────────────────────

def make_cell_2d(x, y):
    traits = ["x", "y"]
    cell = MorphoCell(state_dim=2, traits=traits, noise_scale=0.0)
    cell.state.trait_strengths["x"] = float(x)
    cell.state.trait_strengths["y"] = float(y)
    return cell


def get_positions(cells):
    return np.array([
        [c.state.trait_strengths["x"], c.state.trait_strengths["y"]]
        for c in cells
    ], dtype=np.float32)


# ─── Classifier ───────────────────────────────────────────────────────────────

def logreg_accuracy(X_train, y_train, X_test, y_test):
    from sklearn.linear_model import LogisticRegression
    clf = LogisticRegression(max_iter=500, solver="lbfgs")
    clf.fit(X_train, y_train)
    return float(np.mean(clf.predict(X_test) == y_test)), clf


def logreg_loo(X, y):
    """Leave-one-out accuracy for small datasets (N=4)."""
    from sklearn.linear_model import LogisticRegression
    correct = 0
    for i in range(len(X)):
        mask = np.ones(len(X), dtype=bool); mask[i] = False
        clf = LogisticRegression(max_iter=500, solver="lbfgs")
        try:
            clf.fit(X[mask], y[mask])
            if clf.predict(X[[i]])[0] == y[i]:
                correct += 1
        except Exception:
            pass
    return correct / len(X)


# ─── XOR Experiment ───────────────────────────────────────────────────────────

def run_xor():
    rng = np.random.RandomState(SEED)
    np.random.seed(SEED)

    # XOR corners: 4 classes × XOR_CELLS_CORNER cells with small noise
    xor_corners = [
        (0.1, 0.1, 0),
        (0.1, 0.9, 1),
        (0.9, 0.1, 1),
        (0.9, 0.9, 0),
    ]

    X_raw_list, y_list = [], []
    for cx, cy, lbl in xor_corners:
        for _ in range(XOR_CELLS_CORNER):
            nx = float(np.clip(cx + rng.randn() * XOR_NOISE, 0.0, 1.0))
            ny = float(np.clip(cy + rng.randn() * XOR_NOISE, 0.0, 1.0))
            X_raw_list.append([nx, ny])
            y_list.append(lbl)

    X_raw    = np.array(X_raw_list, dtype=np.float32)
    y_labels = np.array(y_list,     dtype=np.int64)

    # Baseline: LogReg on raw positions (should fail ~0.50 due to XOR)
    from sklearn.linear_model import LogisticRegression
    from sklearn.model_selection import cross_val_score
    baseline_scores = cross_val_score(
        LogisticRegression(max_iter=500), X_raw, y_labels, cv=4
    )
    baseline_acc = float(baseline_scores.mean())

    # Build field + cell_labels
    field = MorphoField(mode="full", neighbors_k=NEIGHBORS_K, noise_scale=0.0)
    cells = []
    cell_labels = {}
    for (nx, ny), lbl in zip(X_raw_list, y_list):
        c = make_cell_2d(nx, ny)
        field.cells_by_dim[2].append(c)
        cell_labels[c.id] = lbl
        cells.append(c)

    rule = FeaturePullInteractionRule(
        cell_labels=cell_labels,
        learning_rate=LEARNING_RATE,
        compression_rate=1.0,
    )

    tracemalloc.start()
    t0 = time.perf_counter()

    for ep in range(XOR_EPOCHS):
        field.propagate_dynamics(rule)
        if (ep + 1) % 40 == 0:
            pos = get_positions(cells)
            # centroid per corner group
            for ci, (cx, cy, lbl) in enumerate(xor_corners):
                grp = pos[ci * XOR_CELLS_CORNER:(ci + 1) * XOR_CELLS_CORNER]
                ctr = grp.mean(axis=0)
                print(f"  [XOR] ep{ep+1}  corner({cx:.0f},{cy:.0f}) label={lbl}"
                      f"  centroid=({ctr[0]:.3f},{ctr[1]:.3f})")

    elapsed_ms = round((time.perf_counter() - t0) * 1000, 1)
    _, peak = tracemalloc.get_traced_memory()
    tracemalloc.stop()

    X_final = get_positions(cells)
    organic_scores = cross_val_score(
        LogisticRegression(max_iter=500), X_final, y_labels, cv=4
    )
    organic_acc = float(organic_scores.mean())

    # Cluster centroids and separation
    centroids = np.array([
        X_final[ci * XOR_CELLS_CORNER:(ci + 1) * XOR_CELLS_CORNER].mean(axis=0)
        for ci in range(4)
    ])
    corner_labels = np.array([lbl for _, _, lbl in xor_corners])
    same_d, cross_d = [], []
    for i in range(4):
        for j in range(i + 1, 4):
            d = float(np.linalg.norm(centroids[i] - centroids[j]))
            (same_d if corner_labels[i] == corner_labels[j] else cross_d).append(d)

    mean_same  = float(np.mean(same_d))  if same_d  else 0.0
    mean_cross = float(np.mean(cross_d)) if cross_d else 0.0
    cluster_ratio = round(mean_cross / max(mean_same, 1e-8), 3)

    print(f"\n  [XOR] baseline={baseline_acc:.4f}  organic={organic_acc:.4f}  "
          f"same_centroid_dist={mean_same:.4f}  cross_centroid_dist={mean_cross:.4f}  "
          f"ratio={cluster_ratio}x")

    return {
        "baseline_logreg_raw":      round(baseline_acc,   4),
        "organic_logreg_final":     round(organic_acc,    4),
        "separability_gain":        round(organic_acc - baseline_acc, 4),
        "mean_same_centroid_dist":  round(mean_same,      4),
        "mean_cross_centroid_dist": round(mean_cross,     4),
        "cluster_separation_ratio": cluster_ratio,
        "final_centroids": {
            f"({xor_corners[i][0]:.0f},{xor_corners[i][1]:.0f})_label{xor_corners[i][2]}":
                centroids[i].tolist()
            for i in range(4)
        },
        "cells_per_corner": XOR_CELLS_CORNER,
        "epochs":           XOR_EPOCHS,
        "elapsed_ms":       elapsed_ms,
        "peak_ram_mb":      round(peak / 1024 / 1024, 4),
    }


# ─── Two Moons Experiment ─────────────────────────────────────────────────────

def make_two_moons(n, noise, rng):
    n_half  = n // 2
    theta_a = np.linspace(0, np.pi, n_half)
    theta_b = np.linspace(0, np.pi, n - n_half)

    Xa = np.stack([np.cos(theta_a),       np.sin(theta_a)],       axis=1)
    Xb = np.stack([1 - np.cos(theta_b),   0.5 - np.sin(theta_b)], axis=1)
    Xa += rng.randn(n_half, 2)      * noise
    Xb += rng.randn(n - n_half, 2)  * noise

    X = np.vstack([Xa, Xb]).astype(np.float32)
    y = np.concatenate([np.zeros(n_half, dtype=np.int64),
                        np.ones(n - n_half, dtype=np.int64)])
    # Normalize to [0.05, 0.95]
    lo, hi = X.min(axis=0), X.max(axis=0)
    X = (X - lo) / (hi - lo + 1e-8) * 0.90 + 0.05
    return X, y


def run_two_moons():
    rng = np.random.RandomState(SEED)
    np.random.seed(SEED)

    X, y = make_two_moons(MOONS_N, MOONS_NOISE, rng)
    split = int(0.7 * MOONS_N)
    X_tr, y_tr = X[:split], y[:split]
    X_te, y_te = X[split:], y[split:]

    baseline_acc, _ = logreg_accuracy(X_tr, y_tr, X_te, y_te)
    print(f"  [Moons] baseline LogReg on raw: {baseline_acc:.4f}")

    # Build field + cell_labels
    field = MorphoField(mode="topological", neighbors_k=NEIGHBORS_K, noise_scale=0.0)
    cells = []
    cell_labels = {}
    traits = ["x", "y"]
    for (px, py), lbl in zip(X, y):
        c = MorphoCell(state_dim=2, traits=traits, noise_scale=0.0)
        c.state.trait_strengths["x"] = float(px)
        c.state.trait_strengths["y"] = float(py)
        field.cells_by_dim[2].append(c)
        cell_labels[c.id] = int(lbl)
        cells.append(c)

    rule = FeaturePullInteractionRule(
        cell_labels=cell_labels,
        learning_rate=LEARNING_RATE,
        compression_rate=1.0,
    )

    tracemalloc.start()
    t0 = time.perf_counter()

    for ep in range(MOONS_EPOCHS):
        field.propagate_dynamics(rule)
        if (ep + 1) % 20 == 0:
            pos = get_positions(cells)
            std = float(pos.std(axis=0).mean())
            print(f"  [Moons] epoch {ep+1}/{MOONS_EPOCHS}  pos_std={std:.4f}")

    elapsed_ms = round((time.perf_counter() - t0) * 1000, 1)
    _, peak = tracemalloc.get_traced_memory()
    tracemalloc.stop()

    X_final = get_positions(cells)
    X_tr_f, X_te_f = X_final[:split], X_final[split:]
    organic_acc, _ = logreg_accuracy(X_tr_f, y_tr, X_te_f, y_te)
    print(f"  [Moons] organic LogReg on final: {organic_acc:.4f}")

    return {
        "n_samples":        MOONS_N,
        "noise":            MOONS_NOISE,
        "baseline_logreg":  round(baseline_acc, 4),
        "organic_logreg":   round(organic_acc,  4),
        "separability_gain": round(organic_acc - baseline_acc, 4),
        "epochs":           MOONS_EPOCHS,
        "elapsed_ms":       elapsed_ms,
        "peak_ram_mb":      round(peak / 1024 / 1024, 4),
    }
